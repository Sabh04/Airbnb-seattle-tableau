# Airbnb-seattle-tableau
Data visualization project in Tableau: Explore Airbnb listings, occupancy, pricing, and reviews in Seattle.

🏠 Airbnb Seattle Analysis – Tableau Dashboard Project

Overview

This project explores Seattle’s Airbnb market using real data on listings, reviews, and calendar availability. An interactive Tableau dashboard visualizes insights on pricing, occupancy, reviews, and host performance, making it ideal for data visualization portfolios, market analysis, or tourism studies.
________________________________________

Project Goal

Understand trends in Seattle’s Airbnb market by analyzing:
•	Listing characteristics: price, amenities, location, property type
•	Review sentiments and scores
•	Availability and pricing dynamics over time
•	Neighborhood performance and host metrics

🔍 Key Insights

•	Listings Overview: 3,800+ listings concentrated in neighborhoods like Capitol Hill, Downtown, and Queen Anne. Average price ~$127/night, with peak pricing in summer.
•	Reviews: High average rating (~94/100). Positive feedback highlights location, cleanliness, and host responsiveness. Common issues include noise and parking.
•	Availability: Peak occupancy during summer. Weekends are 20–30% more expensive than weekdays.
•	Top Findings: Superhosts have 15% higher review scores and occupancy rates. Entire homes dominate (58%) and command premium pricing.
________________________________________

📊 Tableau Dashboard Features

•	Data blending/joins across Listings, Reviews, and Calendar sheets
•	Calculated fields (occupancy rate, average nightly price)
•	Visualizations: Maps, bar/line charts, heatmaps, scatter plots
•	Interactive dashboards with filters, slicers, and parameters
•	Text analysis via word clouds from review comments

▶️ How to Use

1.	Download or clone the repository
2.	Open Tableau Full Project.xlsx in Excel to explore raw data
3.	Open Seattle-Airbnb-Dashboard.twbx in Tableau Desktop or Tableau Public:
o	Connect the Excel sheets as data sources
o	Explore dashboards (Price by Neighborhood, Review Trends, Availability Calendar)
o	Use filters for custom insights
Requirement: Tableau Desktop or Tableau Public (free version)
